NepTrainKit documentation
=========================
NepTrainKit 是一个专注于神经网络势函数（NEP）训练数据集操作和可视化的Python工具包。它主要用于简化和优化 NEP 模型训练过程，提供直观的图形界面和分析工具，以帮助用户更高效地调整超参数并评估模型性能。
